# Lessons

- Keep raw exports and old scaffolds out of normal context; curate compact facts into project artifacts.
- Use ProjectForge-native folders instead of duplicating historical `ai/` folder conventions.
- Prove one real data source before building abstractions for many sources.
- Tests and coherence checks are part of the deliverable, not optional cleanup.
- After each heterogeneous source implementation, record a short Implementation Lessons artifact: confirmed predictions, incorrect predictions, unexpected difficulties, reusable patterns, and future prediction changes.
